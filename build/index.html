Bookmarking things instead of sending them to Pocket

My Pocket queue has about 8000 items in it; which is absurd. Mostly because I started dumping anything into it, not just things I want to read, but resources, other links, etc. I've put a moratorium on my Pocket use until I sort out the archive of links I downloaded; at which point I'll clear it out and start using it correctly. In the meantime I've started dumping things into a messy unsorted bookmarks folder. Just as fun, maybe almost as useful!
